# leaf-map
*  *HojaTrail* is a website where you can find your nearest local book store or public library. It also provides details like price matching for your shopping covenience. It would help give back to the community by giving local businesses and recreational spaces more visivility and consequently more visitors. 



* Origins of the name include: *Trail* suggests discovery, exploration, and movement
*Hoja* adds a unique, cultural touch. 

* For now the slogan is: **Follow the trail to your next read.**

## Features: 
* Based on the user's location to pin point nearest spots. 
* Presents general info of the selected location like business hours. 
* In the future maybe there could be a database for stock items and reviews of outdoor locations. 

## Project Overview:
### 3 pages: 
- Home (Map + Menu) 
- Details of location selected 
- About HojaTrail 

### Elements used: 
- HTML (index.html, store.html, about.html)
- CSS (styling)
- JS (app.js, map.js, cursor.js)
- API (maybe...)

## ABOUT: HOJATRAIL 
Built with a passion for reading, nature and design, HojaTrail is a nature inspired map experience made to help users discover nearby bookstores and libraries and also nearby peaceful spaces where you can enjoy your books or just explore! 

##  GOAL 

The ultimate goal here is to blend technology and nature to inspire exploration.
